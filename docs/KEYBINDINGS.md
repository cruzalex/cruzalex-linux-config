# cruzAlex Linux Keybindings

Complete reference of all keyboard shortcuts. Based on Omarchy defaults.

> **Tip**: Press `Super + K` at any time to view these keybindings in the terminal.

## Core

| Binding | Action |
|---------|--------|
| `Super + Space` | Application launcher (Walker) |
| `Super + Alt + Space` | cruzAlex control menu |
| `Super + Return` | Terminal (Ghostty) |
| `Super + K` | Show keybindings |
| `Super + Escape` | Power menu (lock/suspend/reboot/shutdown) |

## Window Management

| Binding | Action |
|---------|--------|
| `Super + W` | Close window |
| `Ctrl + Alt + Delete` | Close all windows |
| `Super + T` | Toggle floating |
| `Super + O` | Toggle sticky (pin to all workspaces) |
| `Super + F` | Fullscreen |
| `Super + Alt + F` | Full width (keep top bar) |
| `Super + J` | Toggle split direction (horizontal/vertical) |

### Focus & Move

| Binding | Action |
|---------|--------|
| `Super + Left/Right/Up/Down` | Focus window in direction |
| `Super + Shift + Left/Right/Up/Down` | Move window in direction |
| `Super + =` | Grow window right |
| `Super + -` | Grow window left |
| `Super + Shift + =` | Grow window down |
| `Super + Shift + -` | Grow window up |
| `Super + Mouse Click` | Move window |
| `Super + Right Mouse` | Resize window |

### Window Grouping

| Binding | Action |
|---------|--------|
| `Super + G` | Toggle window grouping |
| `Super + Alt + G` | Remove window from group |
| `Super + Alt + Tab` | Cycle windows in group |

## Workspaces

| Binding | Action |

| `Super + Shift + Tab` | Previous workspace |

### Scratchpad
